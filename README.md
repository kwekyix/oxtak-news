# Oxtak Press & Coverage Blog

Static press/coverage page hosted at **oxtak.com/blog**. Shows curated media mentions of Oxtak in a filterable, searchable timeline. No framework — plain HTML, CSS, and JS.

---

## Folder structure

```
oxtblog3/
├── oxtak_scraper.py       # scrapes the web for mentions → oxtak_mentions.json
├── build_blog.py          # review + build pipeline → public/oxtak_data.js
├── oxtak_mentions.json    # raw scraper output (auto-generated, do not edit)
├── oxtak_approved.json    # manually curated list of approved mentions
├── logo/                  # brand assets
└── public/                # upload this entire folder to oxtak.com/blog
    ├── index.html         # page shell — never modified by the pipeline
    ├── style.css
    ├── app.js             # rendering, filtering, search, theme toggle
    ├── oxtak_logo.svg     # brand logo used in nav
    └── oxtak_data.js      # generated by build_blog.py — do not edit manually
```

---

## Pipeline

```
Step 1:  python oxtak_scraper.py           # scrape → oxtak_mentions.json
Step 2:  python build_blog.py --review     # approve / reject each new mention
Step 3:  python build_blog.py              # build → public/oxtak_data.js
Step 4:  upload public/ to oxtak.com/blog
```

### Step 2 — Review prompt keys
| Key | Action |
|-----|--------|
| `Y` | Approve — added to the blog |
| `N` | Reject — discarded |
| `S` | Skip — will appear again next run |

## Adding a mention manually

Skip the scraper and edit `oxtak_approved.json` directly. Each entry follows this shape:

```json
{
  "url":         "https://example.com/article",
  "source":      "Example Media",
  "source_type": "tech_media",
  "title":       "Article title",
  "date":        "2026-06-15",
  "snippet":     "Short excerpt shown on the card.",
  "language":    "EN"
}
```

Valid `source_type` values: `tech_media`, `news`, `blog`, `crowdfunding`, `social_x`, `social_linkedin`, `social_facebook`, `social_reddit`, `video_youtube`

Then run:

```
python build_blog.py
```

---

## Removing a mention permanently

1. Delete its entry from `oxtak_approved.json`
2. Add its URL to `BLOCKED_URLS` in `oxtak_scraper.py` so it never reappears
3. Run `python build_blog.py`

## Design notes

- `index.html` is never touched by the pipeline — only `oxtak_data.js` is generated
- Dark default theme; light mode toggled via `html.light` class + CSS variables
- Theme preference persisted in `localStorage`
- Cards grouped by month, filterable by source type, searchable by title / snippet / source
- Fonts: Space Grotesk (headings), Inter (body) — loaded as variable fonts from Google Fonts
